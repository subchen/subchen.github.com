第三方依赖包 Dependences

  • JDK 1.6+
  • ANTLR runtime 4.x
  • Slf4j 1.7.x

注意： jetbrick-template 需要使用 jdk6+ 里面的 tools.jar 来编译模板生成的 Java 源文件，所有需要完整的 jdk6+ 的运行环境，而不是 jre6+.
