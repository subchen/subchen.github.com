概述 Overview

类似于 Struts, Spring MVC 等经典 MVC 框架，jetbrick-webmvc 通过大量的经验总结，提供小巧、强大，更灵活的 webmvc。

  • 小巧，轻量，易上手
  • 支持 Restful
  • IoC 注入，易管理，易测试
  • Interceptor 机制，功能强大
  • Plugin 机制，易扩展
  • 完全自定义的 Annotation，灵活定制
  • 内置文件上传，下载支持
  • 内置 JSON 支持
  • 内置多种 Result
  • 内置多种 View
