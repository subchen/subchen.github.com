jetbrick 系列开源项目

jetbrick 致力于解决在 Java 开发过程中面临的多种问题。 从前端的 Java 模板引擎，CSS/JS 的 Widget UI 库， 再到后端的 Web MVC，O/R Mapping 等。包含了全生命周期的开发框架。 其中每一个框架都可以单独使用，目前全部项目已经开源，文档正在完善中...

新一代 Java 模板引擎，具有高性能和高扩展性。 适合于动态 HTML 页面输出或者代码生成，可替代 JSP 页面或者 Velocity 等模板。

  • 支持类似与 Velocity 的多种指令
  • 支持静态编译
  • 支持编译缓存
  • 支持热加载
  • 支持类型推导
  • 支持泛型
  • 支持可变参数方法调用
  • 支持方法重载
  • 支持类似于 Groovy 的方法扩展
  • 支持函数扩展

融合了多个项目，文档完善中...

  • jetbrick-commons
  • jetbrick-webmvc
  • jetbrick-orm
